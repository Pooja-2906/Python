This project is a simple Cafe Management System built using Python. It simulates the ordering process in a cafe, allowing customers to choose items from a predefined menu, add them to their order, and calculate the total bill.

Features:

Menu Display: Shows a list of items available for order with their respective prices.

Order Management: Allows customers to select items, check if they are available, and add them to their order.

Total Calculation: Calculates the total price of all items ordered by the customer.
